CubeTwister

Read Me

CubeTwister is your companion to your Rubik's Cube. CubeTwister is a highly configurable editor for cube scripts (aka maneuvers).

Feature Highlights

  • Accurate 3D models of the followig cubes: Rubik's Cube, Rubik's Barrel, Rubik's Diamond, Rubik's Cuboctahedron, Pocket Cube, Revenge Cube and Professor Cube.
    All models can have individual color schemes. The models can even be disassembled. The models are organized per document, multiple cubes per document are supported.
    You can edit the models to reflect your collection of cubes.

  • A configurable parser for all cube notations there are.
    Supports quarter-turn twists, half-turn-twists, rotations, permutations, inversions, reflections, repetitions, conjugations, commutations, and macros.
    The notations are organized per document, multiple notations per document are supported.
    You can define your own notation, and you can translate between notations.

  • A database for scripts, maneuvers and pretty patterns.
    Cubes, Notations, Scripts and Texts are organized in a tree structure.
    The documents are saved in XML format and can be edited using other XML tools.
    You can enter all your scripts and supportive documentation in CubeTwister

  • A sophisticated solver for optimizing your scripts.
    Features a Kociemba Two-Phase Solver (derived from Greg Schmidt's KCube program). The solver generates scripts which are only two or three twists longer than the optimal solution.
    You can enter an arbitrary state of the cube und let it solve. And you can optimise your scripts using the solver.

  • A collection of interactive applets for inclusion in your web site.
    You can integrate virtual cubes and script player applets in your web site. CubeTwister can generate the HTML pages for you, or you can create the HTML code by yourself.
