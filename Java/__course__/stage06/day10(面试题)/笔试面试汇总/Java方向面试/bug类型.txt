1.导出数据重复
大数据导出时发生，这个问题不知道如何导致的， 解决办法 sql里加order by  

2.查询或导出大量数据时页面一直显示加载，直到最后系统没响应也没结果
解决办法，1优化查询sql，2延长会话超时时间，web.xml中session_timeout值加大
还有内存溢出.也可能在计算大量数据时发生，解决办法，大量数据计算时用sql计算，不要用java代码实现

3.JAVA后台主要是数据传输
首先要有一个思路，数据从哪来，到哪去.理清思路后，出现问题，比如前台页面数据不展示.可以看看数据是否是没有传输过来.断点跟踪调试.一步一步的跟数据,如果从后台都返回了数据,那就看看数据是不是输出时出现了问题,比如循环时,el表达式是否正确,大的思路就是数据从哪来到那去,然后断点调试，这就是大体思路。
