【新西兰历险记】透过羊驼的眼，暂别美丽的新西兰

2016-12-09 09:30:42     分类： 旅行--大洋洲

D7，2016年11月17日，游基督城附近的阿卡罗阿小镇及羊驼牧场，制高点上看基督城全貌，晚班飞机飞回中国

已经是离开家门的第七天了，这日子过得还真是如飞云流水般神速，尽管其中掺杂着谁都不想遇到的触目惊心。 基督城旅馆的清晨，大家的节奏基本上都跟着小衣襟一起，她起，
大家都起。 整理好行李，件数还真是不少，但都是小小的一包包，因为大的行李箱已经丢掉了。

行李具体是：

  • 洗漱用品
  • 衣服、鞋子、包包
  • 相机、拍立得(不小心丢了)

D6，2016年11月15日，成功坐直升飞机离开凯库拉，在基督城得到当地华人的热情慰问，满满的感慨和感动

又在房车里睡了一晚，又是起了个早。简单地吃点面包饼干，然后拖着昨天忙乱中整理出来的那些大小包裹，回到集合地。把车子开到警察局，车钥匙交到警察手里，然后又全心全
意地开始了等待，等待直升飞机从基督城方向飞过来。约定的集合时间是早上七点钟，急的就先来了，不急的就推后些，反正要排队，反正今天都能走上。
